# falcon-chemicals
*Falcon Chemicals* — A responsive single-page chemical catalogue website for Monufia, Egypt. Features 44+ products with live search, category/status filters, and quote request system. Built with HTML5, CSS3, and vanilla JavaScript in one self-contained file.

Features
44+ Chemical Products — Laboratory reagents, inorganic salts, acids, and specialty chemicals
Live Search & Filter — Search by name, filter by category (Ammonium, Calcium, Acids, Lead, Potassium, Sodium, Zinc, Other) and availability status
Smart Quote System — One-click "Quote" button auto-fills contact form with product details
Responsive Design — Optimized for all devices from 320px mobile to 1920px+ desktop
Zero Dependencies — Single HTML file, only external resource is Google Fonts (Inter)
Tech Stack
HTML5
CSS3 (Custom Properties, Grid, Flexbox)
Vanilla JavaScript
